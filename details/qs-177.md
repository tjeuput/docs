# QS-177: Link-Handling - Fehlerbehebung

## Kategorie
**Bug Fix** | infoskop-Monitor

## Beschreibung

Ein Problem wurde behoben, das das Öffnen von Links in E-Mail- und Vorlagentexten sowie im Formularlinkbaukasten im lokalen Browser verhinderte. Zudem wurde das Verhalten im Formularlinkbaukasten angepasst.

### Problem

Links konnten nicht im lokalen Browser geöffnet werden in folgenden Bereichen:
- **E-Mail-Texte** (WYSIWYG-Editor)
- **Vorlagentexte** (CK5-Editor)
- **Formularlinkbaukasten**

Zusätzlich war das Verhalten im Formularlinkbaukasten nicht intuitiv - die Vorschau öffnete sich bei unbeabsichtigten Aktionen.

### Lösung

1. **Link-Öffnung repariert**: Links können nun korrekt im lokalen Browser geöffnet werden
2. **Verhalten im Formularlinkbaukasten angepasst**:
   - Links müssen nun direkt angeklickt werden für eine Vorschau
   - Verhindert unbeabsichtigtes Öffnen der Vorschau
